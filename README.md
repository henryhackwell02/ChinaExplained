## Deployment Instructions

### Step 1: Prepare Your GitHub Repository

1. Go to your GitHub repository
2. **DELETE ALL existing files** in the repository
3. Make sure you're starting with a completely clean repository

### Step 2: Upload Files

Upload ALL files from this folder to the **ROOT** of your GitHub repository:
- `index.html`
- `economy.html`
- `politics.html`
- `world.html`
- `styles.css`
- `main.js`
- `vercel.json`
- `.gitignore`
- `README.md`

**IMPORTANT**: Files must be at the root level, NOT in a subfolder!

### Step 3: Vercel Will Auto-Deploy

Once files are pushed to GitHub, Vercel will automatically detect and deploy your site.

## Site Structure

- **Homepage** (`index.html`) - Main landing page with hero, features, and newsletter signup
- **Economy** (`economy.html`) - 6 dummy articles about Chinese economic topics
- **Politics** (`politics.html`) - 6 dummy articles about Chinese political topics  
- **World** (`world.html`) - 6 dummy articles about China's global affairs

## Design Theme

The site uses a China-inspired color palette:
- **Red** (#e63946) - Primary accent color
- **Gold** (#f4a261) - Secondary accent color
- **Jade** (#2a9d8f) - Tertiary accent for economy topics
- **Dark background** with subtle mountain silhouette parallax effect

## Troubleshooting

**404 Error**: Ensure all files are in the root directory of your repository.

**Styles not loading**: Verify that `styles.css` and `main.js` are in the same directory as the HTML files.

**Animations not working**: Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R).
